Link locais,externo e dowlond de arquivos

Os links ou huperlinks são ligações que existem entre páginas.Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu propio site. Acesse uma outra página neste messmo servidor clicando no link anterior.

Também podemos colocar documentos em outros pasta local.Acesse uma página em outra pasta clicando no link anterior.

Também é possivel acessar outro site usando âncoras.Por exemplo, você pode criar um link para acessar o YouTube(não se esqueça de clicar no botão 'VOLTAR'do seu navegador depois, para voltar pra cá).

O link acima faz o usuario ir pra outro site e abandonar o meu.O odeal é que ele abra outro sites em uma nova aba.Acesse agora o YouTube em uma nova aba em branco.

Os links também podem ser usado para o download de arquivos de outro formato como ZIP,PDF,RAR,etc.Por exemplo, abra aqui um livro PDF de exmplo ou baixe a verção compactada do livro.

OBS:Clique sempre em exibir o código fonte para ver como tudo está sendo criado.
